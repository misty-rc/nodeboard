Nodeboard
=========

いろいろやる用。要するに掲示板

Installation
============

    $ git clone https://misty-rc@github.com/misty-rc/nodeboard.git
    $ cd nodeboard
    $ npm install
    $ node app.js

system required
---------------
 * [node.js](http://nodejs.org/)
 * [mongodb](http://www.mongodb.jp/)
 * [redis](http://redis.io/)

-----------

 * [jade]()
